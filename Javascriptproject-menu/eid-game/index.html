Tentang game Eid Blessing Hunter

Game ini merupakan bagian dari project tahunan yang dikerjakan oleh Fikri Mulyana Setiawan dalam rangka memeriahkan hari raya idul fitri setiap tahunnya. Jika pada tahun-tahun sebelumnya project yang dikerjakan merupakan project berbasis desain visual, maka pada tahun ini (2025 - 1446 H) project yang dikerjakan fikri merupakan sebuah game berbasis web yang dapat dimainkan melalui PC ataupun smartphone.

The Game's Rules

Selama permainan, akan ada 4 objek yang akan jatuh, yaitu uang (THR), cookie, petasan dan lalat. Pemain diminta untuk mengumpulkan THR dan cookie sebanyak mungkin sembari menghindari petasan dan lalat. Setiap kali pemain mendapatkan THR, maka skor pemain akan meningkat sebesar 3, sedangkan jika mendapatkan cookie, skor pemain akan meningkat sebesar 1. sebaliknya, jika pemain mendapatkan petasan atau lalat, skor pemain akan berkurang sebesar -4. Pada game ini terdapat sistem level, dimana pemain akan naik level setiap kali mencapai skor kelipatan 5. seiring meningkatnya level, item yang jatuh akan semakin banyak dan semakin cepat, tentunya akan membuat game ini semakin menantang. have fun!

How To Play?

Pada game ini, pemain akan diminta untuk menggerakkan keranjang dengan tombol panah (untuk PC) atau dengan cara menggeser keranjang (pada perangkat touchscreen) dengan tujuan untuk mengumpulkan item yang berguna, yaitu uang (THR) dan cookie.
