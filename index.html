My Programming Journey

September 2018

Kelas 10 SMA. Saat itu adalah saat pertama kali saya tertarik pada bidang programming. saya dikenalkan dengan dunia programming oleh guru saya di sekolah saat kelas TIK (Teknologi Informasi & Komunikasi). saat itu saya diperenalkan dengan bahasa pemrograman Pascal

Oktober 2018 - September 2019

Saya mulai menekuni bahasa pemrograman pascal dan sintaks-sintaks nya, hingga saya bisa membuat berbagai program pascal sederhana seperti program game tebak angka, program penghitung luas bidang, dan sebagainya

Desember 2019

Setelah cukup paham dengan bahasa pascal, saya mempelajari bahasa pemrograman baru, bahasa python. pada waktu itu saya tertarik pada bahasa python karena sintaks nya yang relatif lebih mudah dibanding bahasa lain dan bahasa python juga digunakan dalam ruang lingkup yang luas. saya pertama kali belajar bahasa python menggunakan sebuah buku yang saya beli waktu itu.

Januari 2020 - April 2020

Saya mulai belajar pada bulan Januari karena bulan sebelumnya saya sibuk mempersiapkan Olimpiade Fisika yang saya ikuti. Saya belajar bahasa python dari berbagai macam sumber seperti buku, youtbube dan website - website yang membahas tentang programming.

Mei 2020

Saya mempelajari tentang pemrograman website. Saya pertama kali belajar di website Khan Academy (Thank to Khan Academy) .Dari sana saya belajar banyak tentang pemrograman website. dan dari sana juga lah keinginan saya untuk membangun website sendiri mulai muncul

Juni 2020

Saya membuat akun github saya dan meng-host website pertama saya disana. Website tersebut hanya menggunakan HTML dan sedikit sentuhan CSS. sangat sederhana

Juli 2020 - Agustus 2020

Karena tidak puas dengan website yang sebelumnya, saya membuat website baru, dengan lebih banyak penggunakan style CSS dan juga bahasa pemrograman javascript. Disaat ini pula saya mempelajari bahasa Javascript. Dan disaat yang bersamaan saya mempelajari tentang Python GUI (Graphical User Interface) menggunakan library Tkinter.

Agustus 2020 - Sekarang

Saya mulai menuliskan Blog pada website saya dan terus meng-upgrade style CSS nya. Dan akhir-akhir kni saya menempatkan semua project javascript saya, seperti program Kriptografi sederhana, game, dan kalkulator, sembari terus belajar tentang bahasa pemrograman Python.
